# Introduction

Motorica Studio is a software program that integrates Motorica's animation generation and editing tools inside Unreal Engine 5.

![](/images/motorica-studio-editor-example-1.png)

To get started:

- [Register](/get-started/installing/#register-at-motorica) at Motorica
- [Download](/get-started/installing/#download-motorica-studio) Motorica Studio
- [Install](/get-started/installing/#install-guides) Motorica Studio
- [Configure](/get-started/installing/#configure-motorica-studio) Motorica Studio
